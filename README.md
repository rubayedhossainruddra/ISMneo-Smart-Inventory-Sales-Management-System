ISM neo - Smart Inventory & Sales Management System 

A modern web-based Inventory and Sales Management System built with PHP,
MySQL, and JavaScript to help small businesses, retail shops, and
warehouses efficiently manage products, track sales, generate invoices,
and monitor business performance through an interactive dashboard.

The system includes barcode scanning, generating barcode, automated calculations, sales
tracking, invoice generation, and role-based access control, making it a
complete solution for business inventory management.

  ------------------
  TECHNOLOGY STACK
  ------------------

Frontend: HTML5, CSS3, JavaScript Backend: PHP (Core) Database: MySQL
Barcode: ZXing
Charts: Chart.js Icons: Font Awesome Fonts: Poppins, Roboto,

  --------------
  KEY FEATURES
  --------------

Inventory Management - Add, edit, and delete products - Automatic
product quantity updates - Low stock alerts - Product search and
filtering - Bulk product deletion - CSV export with barcode support

Sales Management - Create multi-product sales orders - Automatic stock
deduction - Real-time price calculations - Sales history and invoice
tracking - Restore stock when sales are deleted

Invoice System - Professional invoice generation - Auto-generated
invoice numbers - Printable invoice templates - Invoice history
tracking - Company information customization

Dashboard Analytics - Total products in inventory - Inventory value
calculation - Today’s sales - Monthly sales - Monthly profit - Low stock
alerts

Includes charts: - Monthly sales charts - Top-selling product pie charts

  --------------------------
  BARCODE SCANNING SUPPORT
  --------------------------

Camera barcode scanning USB barcode scanners Automatic product detection
Beep sound confirmation Mobile vibration feedback

Powered by ZXing barcode library.

  -------------------
  SECURITY FEATURES
  -------------------

-   Password hashing using bcrypt
-   Protection against SQL Injection
-   XSS prevention
-   Input sanitization
-   Secure session management
-   Role-based access control

  ------------
  USER ROLES
  ------------

Administrator - Full system access - User management - System
configuration - Invoice settings - Data reset
functionality

Regular User - Inventory management - Sales processing - Dashboard
analytics

  ------------------------
  MULTI CURRENCY SUPPORT
  ------------------------

Supports 30+ currencies including: USD, BDT, INR, EUR, GBP, JPY, CNY,
PKR, THB, MYR, AUD, CAD, SGD, AED, SAR, KWD, QAR, BHD, OMR, NPR, LKR,
MMK, VND, PHP, IDR, KRW, RUB, BRL, MXN, ZAR, NGN

  --------------------
  DATABASE STRUCTURE
  --------------------

users – User authentication and roles inventory – Product inventory data
sales – Sales transaction records invoices – Invoice details
system_settings – Application configuration activity_logs – System
activity tracking

  ------------------
  ACTIVITY LOGGING
  ------------------

Tracks: - Product additions and updates - Sales creation and deletion -
User actions - Login and logout events
